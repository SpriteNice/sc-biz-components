# 安装

## 环境准备

在开始之前，请确保你的开发环境满足以下要求：

- Node.js >= 16.0.0
- pnpm >= 8.0.0（推荐）或 npm >= 7.0.0
- React >= 18.0.0
- Ant Design >= 5.0.0

## 安装 CLI 工具

### 全局安装（推荐）

```bash
npm install -g create-sc-biz-components
```

或使用 pnpm：

```bash
pnpm add -g create-sc-biz-components
```

### 本地安装

如果你不想全局安装，也可以在项目中本地安装：

```bash
pnpm add -D create-sc-biz-components
```

然后通过 npx 使用：

```bash
npx sc-biz init
```

## 验证安装

安装完成后，运行以下命令验证是否安装成功：

```bash
sc-biz --version
```

如果看到版本号，说明安装成功。
