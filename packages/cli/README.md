# @sc-biz/cli

业务组件库 CLI 工具，快速添加常用业务组件到项目中。

## 安装

```bash
npm install -g @sc-biz/cli
# 或
pnpm add -g @sc-biz/cli
```

## 使用

### 初始化配置

```bash
sc-biz init
```

### 查看可用组件

```bash
sc-biz list
```

### 添加组件

```bash
# 交互式选择
sc-biz add

# 直接添加指定组件
sc-biz add shine-button
sc-biz add use-counter
```
